# 项目简介

一个用于与Gemini 1.5 Pro交互创作故事的应用。

功能：
- 兼容移动端的页面
- 新建故事、设置故事背景
- 在故事下新建角色、设置角色信息、设置角色头像
- 每个故事可以撰写多篇故事
- 为每篇故事单独设置额外的要求（文风、前置剧情等）
- 与模型轮流推进故事
- 编辑、删除任何模型已输出的故事情节（更简单地控制模型接下来的输出）

## 安装和运行指南

### 前端部署

1. 进入前端项目目录：
   ```bash
   cd frontend
   ```
2. 安装依赖：
   ```bash
   npm install
   ```
3. 运行开发服务器（用于开发测试）：
   ```bash
   npm run serve
   ```
4. 生产环境打包：
   ```bash
   npm run build
   ```
5. 根据 `env.example` 文件创建 `.env` 文件，进行配置

### 后端部署

1. 进入后端项目目录：
   ```bash
   cd backend
   ```
2. 安装依赖：
   ```bash
   composer install
   ```
3. 根据 `env.example` 文件创建 `.env` 文件，进行相关配置
4. 运行yii migration：
   ```bash
   yii migrate
   ```
5. 选择你喜欢的Web服务器进行配置

### 备注

- 默认用户token为 `8nD5k1hopW`
- 如果你的云服务器不能访问你需要的模型，可以在 `backend/.env` 中将FRONTEND_PROXY设置为1
  + 此时，前端会代替后端向模型发送生成请求
  + **这会将你的API Key暴露给前端，建议仅在供个人使用时使用此功能**
